Flex Gateway running in Connected Mode enables you to monitor APIs with Anypoint Platform. By using Anypoint Platform, you can configure logs and view key metrics and analytics.

Use the following tasks to get started monitoring Flex Gateway running in Connected Mode:

* xref:flex-conn-message-log.adoc[Configure Message Logging] - Apply a Message Logging policy
to your API instance in API Manager enables you to view basic logs for APIs managed by Flex Gateway
within API Manager.
* xref:flex-use-api-alerts.adoc[Use API Alerts] - Use API alerts for Flex Gateway in Connected Mode, you can add and configure email notifications that flag unusual API behavior.
* xref:flex-view-api-metrics.adoc[View Key Metrics for Flex Gateway APIs] - Collect metrics via API Manager and Anypoint Monitoring for active Flex Gateway APIs. You can view charts related to these metrics and filter the data by date.
* xref:flex-conn-third-party-logs-config.adoc[Configure Flex Gateway Logs for Third-Party Services] - Configure log output for consumption by third-party services such as Splunk. Use a configuration YAML file to configure logs for third-party services.
