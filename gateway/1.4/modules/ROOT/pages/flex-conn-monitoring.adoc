Connected Mode enables you monitor Flex Gateway APIs by using Anypoint Platform.

Get started monitoring Flex Gateway running in Connected Mode with the following tasks:

* xref:flex-conn-message-log.adoc[Configure Message Logging] - Applying a Message Logging policy
to your API instance in API Manager allows you to view basic logs for APIs managed by Flex Gateway
within API Manager.
* xref:flex-use-api-alerts.adoc[Use API Alerts] - Using API alerts for Flex Gateway in Connected Mode, you can add and configure email notifications that flag unusual API behavior.
* View Replica Status - PLACE HOLDER
* View Flex Gateway API Status - PLACE HOLDER
* xref:flex-view-api-metrics.adoc[View Key Metrics for Flex Gateway APIs] - API Manager and Anypoint Monitoring collect metrics on active Flex Gateway APIs. You can view charts related to these metrics and filter the data by date.
* xref:flex-conn-third-party-logs-config.adoc[Configure Flex Gateway Logs for Third-Party Services] - Anypoint Flex Gateway enables you to configure log output for consumption by third-party services such as Splunk. Logs for Third-Party services are configure via a configuration YAML file.
