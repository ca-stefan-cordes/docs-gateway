The *Basic Authentication: Simple* policy protects an API by forcing applications to provide a username and password when making requests.

== Definition

----
- policyRef:
    name: http-basic-authentication-flex
  config:
    username: <string> // REQUIRED
    password: <string> // REQUIRED
----

== Example

----
- policyRef:
    name: http-basic-authentication-flex
  config:
    username: chris
    password: admin
----

Example authentication of a request made via `curl`: 

----
curl -v -u chris:admin http://localhost:8080/api/users/1
----
